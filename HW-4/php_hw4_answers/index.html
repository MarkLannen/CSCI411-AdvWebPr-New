Answers for Homework 4

  1. Q: What are the different methods of concatenation and to what data types (and literals) do they apply?

    A: There are 2 concatenation methods for strings.

    • The "." operator concatenates two strings together. Returns concatenation of right and left argument.
    • ".=" appends the argument on the right to the argument on the left.

  2. Q: Describe at least three different ways to declare and initialize an array

    1. Array function: $fruit = array('apples', 'oranges', 'bananas');
    2. Arrays that are automatically indexed: $fruits = ['apples', 'oranges', 'bananas];
    3. Associative arrays: $fruit = array("1"=>"apple", "2"=>"oranges", "3"=>"bananas");

  3. Q: How do you store the results of print_r() to a variable?

    A: Call the built-in function 'print_r and pass in a variable as an argument.

    • $fruit = 'apple';
      print_r($fruit);

  4. Q: What is the var_dump function? How is it different than print_r?

    A: var_dump is a built in function that prints an argument passed to it, along with the arguments type and value.
    print_r just displays the content of the argument, i.e., string, array etc. and does not include type or value information.

  5. Q: What is dynamic typing? Describe some advantages and disadvantages

    A: If a programming language is dynamically typed, it means that a variables data type is checked during run-time as opposed to when being compiled. An advantage is that the programming process can be sped up somewhat as the programmer does not have to manually declare data types for all variables. A disadvantage is that if there is a type-error, it won't be caught until the program is trying to run, rather than when it is being compiled.
